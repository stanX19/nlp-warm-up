# NLP Tutorial Warm-Up Notebook

A small notebook to warm up/check the NLP environment before running for code in UM's NLP course.

It preloads common NLP packages, datasets, and models used across the tutorials. It also contain codes that acts as a quick test to make sure the runtime is ready.

Time saved in testing:

* Google Colab: ~4 minutes
* Local JupyterLab: ~20 minutes

Colab currently uses Python 3.12, and this notebook handles that environment nuance.

Use this before tutorials, revision, exams, or running the notebooks on a new setup.

The notebook: [warm_up.ipynb](warm_up.ipynb)
